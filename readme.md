# Connect 4 🟠🟡

A fun browser-based Connect 4 game built with HTML, CSS, and JavaScript.

## Features
- 2 Player local multiplayer
- vs AI mode (minimax with alpha-beta pruning)
- Score tracking across rounds
- Kid-friendly orange & yellow theme

## How to Play
1. Download all 4 files into the same folder
2. Open `index.html` in any browser
3. No server or install needed

## Files
| File | What it does |
|------|-------------|
| `index.html` | Page structure |
| `style.css` | All styling |
| `game.js` | Game logic & UI |
| `ai.js` | Minimax AI engine |

## How to Win
Drop your coloured piece into a column and be the first to get **4 in a row** — horizontally, vertically, or diagonally.
